> *Event-driven platform for flight delay analytics and reliability scoring using live FAA & OpenSky datasets*

**Stack:** `Java 17` · `Spring Boot` · `Python/FastAPI` · `Apache Kafka` · `PostgreSQL` · `MongoDB` · `React/TypeScript` · `Docker` · `Kubernetes (AKS/EKS)` · `GitHub Actions` · `Azure DevOps` · `Prometheus` · `Grafana` · `JWT` · `GraphQL`

- Ingests live FAA and OpenSky flight data via a FastAPI ingestion service; routes events through Kafka topics to Spring Boot microservices for delay, route, and reliability analytics
- Dual-database architecture — **PostgreSQL** for KPI analytics with optimized indexes, and **MongoDB** for dynamic route configuration
- Secured REST and GraphQL APIs with **JWT/OAuth2** and TLS ingress; RBAC enforced across all microservice boundaries
- Deployed on **Azure AKS** (AWS EKS compatible) via multi-stage GitHub Actions + Azure DevOps CI/CD with Kustomize environment promotion
- Full observability — **Prometheus + Grafana** via Micrometer instrumentation with SLA/SLO dashboards across all services
